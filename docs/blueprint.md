# **App Name**: VersaStack Portfolio

## Core Features:

- AI-Powered Recommendations: Utilize a generative AI tool to suggest projects to showcase based on a skills and technologies detected from user's resume/profile, ensuring the portfolio highlights the most relevant work.
- Interactive Project Showcase: Enable interactive demonstrations of projects directly within the portfolio, providing potential employers with hands-on experience of your skills.
- Dynamic Skill Visualization: Present your technical skills through interactive charts and graphs that automatically update as you add new projects and experiences.
- Automated Code Snippet Integration: Allow for the seamless integration of code snippets from GitHub or other repositories, ensuring that your portfolio always showcases your most recent work.
- Personalized Contact Form: Create a contact form that tailors inquiries based on the projects and skills highlighted in your portfolio, making it easier for potential employers to reach out with relevant opportunities.

## Style Guidelines:

- Primary color: Electric green (#00FF88) to convey a modern, tech-forward aesthetic.
- Background color: Deep navy (#0A0E27) to provide a sophisticated dark mode base.
- Accent color: Cyber blue (#00D4FF) to highlight interactive elements and call-to-actions, adding visual interest.
- Body text: 'Inter', a sans-serif font, will ensure readability and a clean, modern look.
- Code snippets: 'JetBrains Mono', a monospaced font, will ensure the proper formatting and display of code samples.
- Implement a component-based design, visually emphasizing the separation of concerns, modularity and system-level architecture comprehension expected of a full stack developer. This includes consistent spacing and alignment.
- Use smooth, professional transitions throughout the portfolio to showcase your mastery of frontend technologies.
